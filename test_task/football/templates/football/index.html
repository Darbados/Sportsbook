    Here I'll describe in few steps what you can do with this simple site.

  1. By selecting Leagues you'll see all the leagues that we have within our storage.
  2. By selecting Teams you'll see all the teams that we have within our storage.
  3. By selecting Prematch you'll see all the events that we have within our storage, which are about to start later today or in the next days.
  4. By selecting Live you'll see all the live events that we have, but there could be nothing as well, if there are no live events.
  5. By selecting Finished you'll see already finished events with some info for each one.
